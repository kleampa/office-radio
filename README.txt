Instalare
Se uploadeaza pe un server in asa fel incat sa fie disponibil prin intermediul unui browser si accesibil pentru toti cei din birou. Noi l-am pus intr-un anume folder al site-ului companiei.
Se seteaza permisiuni 777 pentru fisierele .txt din folderul bd/

Folosire
1. Cel care are acces la boxe trebuie sa acceseze adresa unde a fost uploadat scriptul dar cu terminatia ?server=1 (ex: www.site.ro/radio/?server=1)
2. Ceilalti colegi din birou doar acceseaza adresa unde a fost uploadat scriptul (ex: www.site.ro/radio/)

Sub nicio forma sa nu fie 2 persoane care intra in acelasi timp cu terminatia ?server=1 pentru ca da peste cap ordinea de redare a melodiilor.
